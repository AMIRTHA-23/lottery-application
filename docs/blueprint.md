# **App Name**: SMSWIN Lottery

## Core Features:

- User Registration & Login: Secure user registration and login functionality.
- Lottery Number Purchase: Allow users to select and purchase lottery numbers for single, double, three, and four-digit games.
- Payment Gateway Integration: Integrate secure payment gateways supporting UPI, cards, and net banking.
- Result Display: Display daily and weekly lottery results.
- Wallet Management: Enable users to manage their in-app wallet balance.
- Admin Dashboard: Provide an admin dashboard with analytics on user purchases and the ability to manage user accounts and lottery events.
- Real-time winner updates: If the drawing has recently completed, generate messages using a tool to determine potential matches for lottery numbers that a user has purchased in the past.

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2) to inspire trust and confidence in the app.
- Background color: Light gray (#F5F5F5) for a clean and modern interface.
- Accent color: Energetic green (#90EE90) to indicate success, particularly with payouts and winnings.
- Body and headline font: 'Inter' sans-serif, for a clean and neutral style suited to either headlines or body text.
- Use simple, clear icons to represent different lottery types and actions.
- Maintain a clean, intuitive layout with clear sections for each feature.
- Subtle animations for displaying results and updating wallet balances.
